# Panda CSS

## Overview

Use Panda CSS as the exclusive styling system for this frontend.

## Guidelines

- Read `references/panda-css.md` before writing styles.
- Prefer recipes and patterns over ad-hoc styles.
- Use tokens and semantic tokens instead of raw values.
